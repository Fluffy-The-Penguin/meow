Game rules and mode explanations

A quick guide to the main game families in Meow Bot and how they work.

Quick Start Picks

Best daily routine

Use /daily, /quests, and /dailywordle together to create repeatable member activity.

Best for active chat

Run /trivia, /truthordare, /wouldyourather, /nhie, and /paranoia when the server already has momentum. Never Have I Ever and Paranoia support public next-prompt buttons.

Best chaos combo

Mix /roast, /timetravel, /anonymousstatement, and social prompts for identity-heavy server moments.

Best long-term loop

Use pets, achievements, quests, counting, and economy systems together so members keep making progress outside of one-off sessions.

Trivia

  • /trivia opens a setup flow for mode, rounds, timing, category, and auto-start.
  • Hosts choose multiple choice or single answer mode, then set difficulty, rounds, timer, category, and auto-start.
  • Players answer in chat, and fast correct answers earn coins and placement bonuses.

Riddles

  • Players type text answers directly into chat.
  • Attempts are limited.
  • Correct solves and top placements are rewarded.

Hangman and scramble

  • Hangman uses letter or full-word guesses.
  • Scramble challenges users to quickly solve jumbled words.
  • Difficulty settings influence challenge and reward level.

Word chain

  • Each response must begin with the last letter of the previous valid word.
  • Dictionary validation prevents fake entries.
  • Turn timers eliminate stalled players.

Wordle and Daily Wordle

  • Players have six attempts to solve the hidden word.
  • Themed, random, and listed themes are supported.
  • Daily Wordle runs one shared word per server per day.

Quiz and number games

  • /countrygame supports country clues, flags, capitals, progressive hints, and population comparisons.
  • /wtp uses Pokemon silhouette rounds with setup, start, and stop flows.
  • /guessthenumber starts a one-off number round in the current channel.
  • /guesschannel keeps a dedicated persistent channel-based guess flow.
  • /guessthenearest runs a multiplayer nearest-number match with host controls and round scoring.

Would you rather and truth or dare

  • Social prompt-driven modes intended to keep conversation moving.
  • Can use generated or chosen options depending on the command.

Social prompts

  • Would You Rather and Truth or Dare keep conversation moving when the server already has momentum.
  • Never Have I Ever and Paranoia include public Next buttons that keep the selected intensity level.
  • Dad Joke and anonymous statement flows work well as lighter social breaks between larger games.

Blackjack, PvP, and social challenges

  • /blackjack provides a coin-based casino-style game against the dealer.
  • /pvp covers direct user-vs-user coinflip, dice, and RPS challenges.
  • /roast gives a quick fun-mode roast when the server wants lighter social chaos.

Pets, lotteries, events, and counting

  • Pets form a long-term progression system with care, mood, jobs, battles, and evolution gates.
  • Lotteries support both admin weekly jackpots and host-funded event lotteries with public buttons and visual dashboards.
  • Server-wide XP and coin events, plus a managed counting channel, create passive activity outside direct game sessions.

FAQ

Common setup and usage questions

Showing all FAQ answers.

Why are slash commands not appearing?

Discord command registration can take a little time after invite. Re-inviting with the proper application command scope or waiting for propagation usually fixes it.

Why is the bot not replying in chat?

Check whether a talk channel is configured, whether the bot can send messages there, and whether you are using the chat features the way they are set up to work.

Why are some games not working?

Some game modes need active channels, enough participants, or no other active game already running in that channel. Others depend on setup commands such as dedicated game channels.

Where did the old text commands go?

Public ! shortcuts are retired. Use slash commands instead, or run /textcommands to see current replacements for common media and quick-fun commands.

How do pets and progression connect?

Coins, quests, achievements, pets, and shop systems reinforce each other. Members earn through activity, then reinvest in collection and progression loops.

How do I reset a broken user state?

Use the administrative reset command if you have access, and be careful to target only the fields you intend to clear.

Where should I start as a new server?

Open /admindashboard if you have Manage Server, or set talk, pet, logs, event, Daily Wordle, anonymous, and counting channels with slash commands. Confirm slash commands are available, then expand into pets, PvP, and larger progression systems.

Who can open the admin dashboard?

Users with Manage Server, administrator, or owner-level authority can use /admindashboard and the web dashboard for servers they can manage.

Why is my server missing from the dashboard?

Sign in with the Discord account that can manage that server, make sure the bot is still in the server, and confirm your server permissions include dashboard access.

What can the admin dashboard configure?

The dashboard can configure supported server channels such as talk, pet, logs, announcements, events, Daily Wordle, anonymous statements, and counting.

Why do some commands fail in DMs?

Server-specific commands need a guild context, permissions, and saved server settings. Use those commands inside a Discord server instead of direct messages.

How do I set a talk channel?

Use /talkchannel or the admin dashboard. The talk channel is where Meow Bot is intended to respond naturally to normal chat.

Will Meow Bot reply everywhere?

No. It is designed around configured talk channels, direct mentions, and supported interactive features rather than replying to every message in every channel.

How do I stop AI replies?

Remove or disable the talk channel with /talkchannel, adjust permissions, or avoid using bot mentions where AI replies are expected.

How do I clear conversation memory?

Use /clearmemory to clear saved conversation memory connected to your bot chat experience.

How do I control my data participation?

Use /mydata for available personal data and participation controls, including opt-out choices supported by the bot.

What message content does the bot read?

Message content is used for features that need typed input, such as talk-channel replies, game answers, counting, reaction triggers, and active channel-based games.

How do automatic reactions work?

Automatic reactions respond to supported message patterns when enabled. Admins can adjust reaction categories, cooldowns, density limits, muted periods, and rude reaction behavior with /reactions.

How do I make reactions less noisy?

Use /reactions to disable categories, add cooldowns, reduce density, mute reactions temporarily, or turn off rude reaction types.

How do I get animal images?

Use /animals for random animal images or pick a supported animal type with autocomplete. Use /cat and /dog for direct cat or dog image commands.

Why does /animals not show every animal?

The bot only offers supported image-focused animal sources that pass filtering. Some animals or formats may be excluded if reliable safe images are not available.

Why did I get a repeated animal image?

The bot tracks used animal image URLs for normal server use, but repeats can still happen when external sources return duplicates, a queue refills, or DM usage is not persisted.

Can /animals return videos?

No. The animal command is image-focused. Video posts and non-image posts are filtered out before the bot sends a result.

How do dog breed choices work?

/dog supports breed autocomplete. Start typing a breed or sub-breed and choose one of the available suggestions.

Why did a media command fail?

Media commands depend on external APIs and safe image sources. If a provider is slow, unavailable, rate limited, or returns unsafe content, the bot may skip or fail that request.

How do movie recommendations work?

/movie uses TMDB-backed discovery, watchlists, ratings, history, feedback, polls, comparisons, and taste overlap features when the movie API is available.

Why are movie results unavailable?

Movie features require a working TMDB connection. If the API key or provider is unavailable, movie recommendations may be disabled until the service is restored.

How do anime commands work?

/anime supports info, recommendations, library browsing, public AniList imports, and character lookup through AniList-backed data.

How do manga commands work?

/manga supports info, recommendations, library browsing, and public AniList imports for reading lists and discovery.

Does AniList import need my password?

No. The bot imports public AniList list data by username and selected list type. Do not share passwords or private account credentials.

How does /identity work?

/identity lets users set, view, or clear optional server-only identity card details such as nickname, DOB, gender or pronouns, country, height, vibes, spirit animal, and selected friends.

Is my identity card global?

No. Identity cards are stored per server. Details set in one server are not meant to become a global profile for every server.

Can I clear identity card fields?

Yes. Use /identity clear and choose the field you want to remove, or clear the whole card if you no longer want it stored for that server.

How do Wordle games work?

/wordle starts a normal Wordle round with configurable mode, word length, and theme options. /dailywordle runs the daily shared challenge.

How do Daily Wordle announcements work?

Admins can set a Daily Wordle notification channel with /dailywordlechannel or the admin dashboard so new daily rounds can be announced in the server.

What trivia modes are available?

/trivia supports multiple-choice and single-answer modes. Hosts can configure rounds, timing, difficulty, categories, and auto-start settings through the setup flow.

Why did trivia not accept my answer?

Check whether the round is still active, whether it is single-answer or multiple-choice mode, and whether your answer format matches the current mode.

How do riddles work?

Riddle rounds accept typed answers in chat, limit attempts, and reward correct solves and placements during the active answer window.

How does Hangman work?

/hangman supports letter guesses or full-word guesses, with optional difficulty and solo or multiplayer modes.

How does Scramble work?

/scramble gives jumbled words for players to solve quickly, with configurable rounds, difficulty, and time per round.

How does Word Chain work?

/wordchain requires each valid word to start with the last letter of the previous valid word. Dictionary validation helps reject fake words.

What modes does /countrygame include?

/countrygame includes country guessing from clues, flag guessing, capital matching, progressive hints, and population comparison rounds.

How does WTP work?

/wtp runs Who's That Pokemon silhouette rounds. Use setup to choose the channel, start to begin, and stop to end the flow.

How do guess-the-number games work?

/guessthenumber starts a one-off number game in the current channel. /guesschannel creates a persistent channel-based guessing flow with limits and cooldowns.

How does Guess The Nearest work?

/guessthenearest starts a multiplayer nearest-number match where players submit guesses and scoring favors the closest answers.

How do I stop an active game?

Use /stopgame for supported active game sessions in a channel, or use game-specific stop commands such as /stopguess or /wtp stop.

How do social prompt commands work?

/truthordare, /wouldyourather, /nhie, and /paranoia generate prompts meant to keep active conversations moving.

Who can press Next on prompts?

Utility Next buttons for supported prompt commands are public, so anyone can continue the flow without needing the original command user.

How do anonymous statements work?

/anonymousstatement sends a server-routed anonymous message to a target using the configured anonymous channel and selected mood style.

How do I set up counting?

Use /counting setup or the admin dashboard to choose a counting channel. The bot tracks the current number, high score, participants, goals, and saves.

Why did counting reset?

Counting can reset when someone posts the wrong number or the same person counts twice in a row and no saves are available.

Can counting be moved without losing progress?

Use /counting move to move counting to a new channel without resetting the active count. Changing the counting channel from dashboard setup may reset active state.

What are counting saves and goals?

Counting saves can protect a run from mistakes while available. Counting goals let servers set reward milestones for a shared counting run.

How do I start with pets?

Use /pet to open the pet panel. Set a pet channel with /petchannel if you want egg events and pet-channel activity in a dedicated place.

How do pet egg claims work?

Daily pet eggs appear as claim buttons in the configured pet channel. Use the button flow rather than old text claim shortcuts.

Can I upload a custom pet image?

Use /petimage to upload or clear an image for your current pet, subject to Discord attachment limits and bot validation.

What happens if I neglect my pet?

The pet system includes care timers, mood, progression consequences, and ghost-pet cleanup. Use /petfuneral if you need to release a ghost pet and move on.

How do pet fights work?

How do tournaments work?
